## Tech Stack

React 18, TypeScript, Vite 5, Tailwind CSS, shadcn/ui, React Router DOM 6, Lucide React, Sonner

## Getting Started

```bash
git clone https://github.com/gemachistesfaye/Haramaya-Gym.git
cd Haramaya-Gym
npm install
npm run dev
```
